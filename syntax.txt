How to get health of cluster and nodes :
	- GET _cluster/health
	- GET _nodes/stats


Create : 
	- POST Name_of_Index/_doc { "field": "value" }
		○ Autogenerated document ID
		○ Will overwrite data
		○ POST favorite_candy/_doc
		○ {
			"first_name": "Lisa",
			"candy": "Sour Skittles"
		}
	- PUT Name_of_Index/_doc/Assigned_ID
		○ Document ID is assigned
		○ Will overwrite data
		○ PUT favorite_candy/_doc/1
		{
			"first_name": "John",
			"candy": "Starburst"
		}
	- PUT Name_of_Index/_create/Assigned_ID
		○ Does not overwrite data
		○ PUT favorite_candy/_create/1
		{
			"first_name": "Finn",
			"candy": "Jolly Ranchers"
		}
Read :
	- GET Name_of_Index/_doc/Document_ID
		○ Retrieves content of the document
		○ GET favorite_candy/_doc/1
Update : 
	- POST Name_of_Index/_update/Document_ID
		○ Updates fields in the document
		○ POST favorite_candy/_update/1
		{
		   "doc": {
			"candy": "M&M's"
		   }
		}
Delete : 
	- DELETE Name_of_Index/_doc/Document_ID
		○ DELETE favorite_candy/_doc/1
