Install Elasticsearch and Kibana locally:
	- Download Elasticsearch from this link.
	- Download Kibana from this link.
	- Move both from downloads to desktop.
	- Extract both .tar.gz files to desktop.
	- Open terminal and CD into extracted Elasticsearch folder.
	- Run bin/elasticsearch
	- Open a new terminal and run:
	- curl --cacert config/certs/http_ca.crt -u elastic https://localhost:9200
		○ Run this if you need to change the password for user [elastic]:
		○ bin/elasticsearch-reset-password -u elastic
	- Enter the password and should get a successful ping to the node:
	- 
	- Then cd into kibana folder and run bin/kibana
		○ Should show the localhost port and url
		○ Go to that url in a browser
	- To configure elastic you have to have an authentication key
		○ And then the verification key will show up in the kibana terminal
	

Create : 
	- POST Name_of_Index/_doc { "field": "value" }
		○ Autogenerated document ID
		○ Will overwrite data
		○ POST favorite_candy/_doc
		○ {
			"first_name": "Lisa",
			"candy": "Sour Skittles"
		}
	- PUT Name_of_Index/_doc/Assigned_ID
		○ Document ID is assigned
		○ Will overwrite data
		○ PUT favorite_candy/_doc/1
		{
			"first_name": "John",
			"candy": "Starburst"
		}
	- PUT Name_of_Index/_create/Assigned_ID
		○ Does not overwrite data
		○ PUT favorite_candy/_create/1
		{
			"first_name": "Finn",
			"candy": "Jolly Ranchers"
		}
Read :
	- GET Name_of_Index/_doc/Document_ID
		○ Retrieves content of the document
		○ GET favorite_candy/_doc/1
Update : 
	- POST Name_of_Index/_update/Document_ID
		○ Updates fields in the document
		○ POST favorite_candy/_update/1
		{
		   "doc": {
			"candy": "M&M's"
		   }
		}
Delete : 
	- DELETE Name_of_Index/_doc/Document_ID
		○ DELETE favorite_candy/_doc/1

Elasticsearch is a search and analytics engine that allows you to store, search, and analyze your data. 

Coupled with Kibana (a UI designed to visualize and explore data in Elasticsearch), you can design a fantastic search experience for your app users!
